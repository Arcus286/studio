# **App Name**: TaskFlow

## Core Features:

- Task Creation: Allow users to create new tasks with a title and detailed description.
- Task Listing: Display all tasks in a list format, showing key details like title and status.
- Status Updates: Enable users to update task statuses (To Do, In Progress, Done).
- Task Details: Provide a detailed view for each task, showing all relevant information.
- Smart Status Suggestion: AI-powered tool suggests the most appropriate status for a task based on its description and recent activity.

## Style Guidelines:

- Primary color: Professional blue (#3498DB) to convey trust and reliability.
- Background color: Light gray (#ECF0F1) to provide a clean and modern backdrop.
- Accent color: Soft gray (#95A5A6) for subtle highlights and secondary elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern and neutral look.
- Use minimalistic icons to represent task status and actions.
- Maintain a clean and organized layout for easy navigation and readability.
- Incorporate subtle animations to provide feedback on user interactions.
